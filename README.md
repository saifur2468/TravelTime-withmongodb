# 🌍 Tourism Management Website

A full-stack Tourism Management web application where users can explore tourist spots from different countries, view detailed information, and manage tourism data. This project is built using modern web technologies.

## 🚀 Live Website

🔗 Live Link: https://travel-web-c6145.web.app/

## 📌 Features

* 🌏 Explore tourist spots by country
* ⭐ View top rated tourist destinations
* 📍 Detailed tourist spot information
* 🔐 User authentication system
* ➕ Add new tourist spots
* ✏️ Update and manage spots
* 🗑️ Delete tourist spot data
* 📱 Fully responsive design
* ⚡ Fast loading UI

## 🛠️ Technologies Used

### Frontend

* React.js
* React Router
* Tailwind CSS
* DaisyUI
* React Icons

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* Firebase Authentication

### Deployment

* Firebase Hosting
* Vercel / Render (Server)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/your-repository-name.git
```

### 2️⃣ Go to project folder

```
cd your-repository-name
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the project

```
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file and add:

```
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
DB_USER=your_database_user
DB_PASS=your_database_password
```
